# NestedIfAlternative

### Simple Java Rule Engine

This lightweight rule engine uses functional programming to reduce nested if-else statements.

### RuleEngine
Purpose of this class is to tie up the rules with the processing logic

### ProcessingLogic
Purpose of this class, is to transfer the logic executed inside the if Block into FUNCTION
And apply that logic on demand when if clause (defined in FilteringCondition) is true

### FilteringCondition
Purpose of this class to define all the conditions used inside if clause

### Params
This is a PoJo where the parameters used in if clause evaluation are bundled together
